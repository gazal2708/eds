What is Bring Your Own Markup?

Edge Delivery Services is independent of the authoring tooling and supports multiple content sources. This means you could provide your own content source and publish the content from a repository you already have to AEM without having to migrate the content first.

The API that enables it is called Bring Your Own Markup (BYOM). It uses HTML as the standard data format.

Getting Started

To implement BYOM, your service needs to return HTML responses that follow the Edge Delivery Services semantic structure. Each page should include:

  • Sections: Each <div> directly under <main> represents a section
  • Blocks: Divs with class names become blocks (e.g., class="hero")
  • Default Content: Regular HTML elements without block classes
  • Metadata: Page metadata via HTML meta tags or metadata block
