How BYOM Overlay Works

Step 1: Configure BYOM overlay via Configuration Service API

Step 2: Create HTML files in GitHub with EDS semantic structure

Step 3: Push to GitHub and reconfigure overlay

Step 4: Preview pages and they're live!

Key Benefits

  • Developer Workflow: Use Git, IDEs, and version control
  • Instant Updates: Push to GitHub and reconfigure
  • Hybrid Approach: Keep Google Drive for some content, use BYOM for others
  • No Migration Required: Overlay existing content sources
  • Full HTML Control: Complete control over markup and structure
